Política de Privacidade - Key BOT

Última atualização: 06 de julho de 2026

Esta Política de Privacidade explica como o Key BOT coleta, utiliza e armazena informações necessárias para o funcionamento de seus recursos no Discord.

1. Dados coletados

O Key BOT pode armazenar apenas os dados necessários para seu funcionamento, incluindo:

- IDs de usuários do Discord;
- IDs de servidores;
- IDs de cargos;
- IDs de canais;
- Chaves (Keys) geradas e utilizadas pelo sistema;
- Configurações relacionadas ao funcionamento do bot em cada servidor.

O Key BOT não coleta senhas, e-mails, mensagens privadas, conteúdo de mensagens ou qualquer informação pessoal além daquela disponibilizada pela API oficial do Discord e necessária para suas funcionalidades.

2. Finalidade dos dados

Os dados são utilizados exclusivamente para:

- Gerenciar o sistema de chaves;
- Conceder e remover cargos temporários;
- Armazenar configurações do servidor;
- Registrar ações relacionadas ao funcionamento do bot;
- Fornecer suporte às funcionalidades oferecidas.

3. Compartilhamento

Os dados armazenados não são vendidos, compartilhados ou disponibilizados a terceiros, exceto quando exigido por lei.

4. Segurança

São adotadas medidas razoáveis para proteger os dados armazenados. Entretanto, nenhum método de armazenamento ou transmissão é totalmente seguro.

5. Exclusão de dados

Administradores do servidor podem solicitar a exclusão dos dados armazenados referentes ao seu servidor.

6. Alterações

Esta Política de Privacidade poderá ser atualizada periodicamente. A versão mais recente será sempre a válida.

7. Contato

Caso tenha dúvidas sobre esta Política de Privacidade, entre em contato por meio do servidor oficial de suporte do Key BOT.
